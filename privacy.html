Mayer Privacy Policy

Privacy Policy

The developers ("we") of this Discord App ("Mayer", "bot") collect data only required to provide specific functions;
we do not share any data nor use it for any other purpose.
Specified data is only collected when a person ("user") uses any of the features ("command", "commands") listed below:

  • gacharoll command:
    • User ID
    • Number of "rolls"
  • tagcalc commands:
    • User ID
    • Selected in-game server region
  • newsfeed, clock, botlang, gamedata commands:
    • Guild ID
    • Channel ID
  • quiz command:
    • User ID
    • Display name/Username
    • Number of "correct answers"
    • Date of the game
Any other data such as command options, "name" or "avatar" etc. is processed temporarily and is not permanently stored.

Data Removal

To have your guild ("server") removed from Mayer's database, use the "remove" command;
to remove your personal data, use the "removemydata" command.
